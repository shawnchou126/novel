# 人物塑造规则

## 🎭 核心原则

**每个重要角色都是有血有肉的人，不是工具人！**

### 0. 主角能力可以是设定

**林深的技能无需反复解释**：
- ✅ 他精通商业策划、营销、谈判 → **这是设定，直接用**
- ❌ 不需要每次都解释"他在大学学过"、"他看过相关书籍"
- ✅ 展现能力时自然流畅即可

**⚠️ 但是！能力设定后不能随意变动**：

**禁止的突然新能力**：
```
❌ 错误示例：
第5章：林深精通商业策划
第10章：林深突然会编程、黑客技术
第15章：林深又突然精通金融衍生品

→ 这是能力膨胀！不合理！
```

**允许的能力成长**：
```
✅ 正确示例：
第5章：林深擅长商业策划（已有能力）
第10-12章：林深通过学习掌握了基础财务分析
→ 展示了学习过程、花费时间
→ 而且是相关领域的自然延伸
```

**能力边界原则**：

1. **已设定能力**：直接使用，无需解释
   - 商业策划 ✅
   - 营销推广 ✅
   - 谈判技巧 ✅

2. **相关延伸能力**：可以快速掌握
   - 市场分析 ✅（商业策划的延伸）
   - 品牌定位 ✅（营销的延伸）

3. **全新领域能力**：需要学习过程
   - 编程、黑客 ❌（需要长期学习）
   - 金融衍生品 ❌（需要专业训练）

4. **绝对禁止**：超能力化
   - ❌ 突然精通所有领域
   - ❌ 成为全知全能的"龙傲天"

**详见**：`novel/linshen-abilities.md`（完整能力设定文档）

## ✅ 必须遵守的规则

### 1. 性格一致性

**禁止突兀变化**：
- ❌ 上一章还温柔体贴，下一章突然冷酷无情
- ❌ 前面怕死，后面突然勇猛无畏
- ❌ 一直理性，突然冲动行事

**允许的变化**：
- ✅ 有明确事件触发（被背叛、受刺激、顿悟等）
- ✅ 经历时间沉淀的成长
- ✅ 在合理范围内的情绪波动

### 2. 变化必须有原因

**如果角色性格/态度发生变化，必须满足以下之一**：

1. **当场发生了重大事件**
   ```
   沈云溪对林深的态度从怀疑→信任
   → 必须有具体事件：林深帮她解决了危机
   ```

2. **埋下伏笔，后续解释**
   ```
   第5章：沈云瑶突然对林深态度变冷
   → 第8章回应：她发现了林深隐瞒的事情
   ```

3. **时间跨度带来的成长**
   ```
   三个月后，林深变得更加成熟
   → 合理，因为时间足够
   ```

### 3. 背景一致性

**角色的过去必须前后呼应**：

❌ **错误示例**：
```
第3章：沈云溪说自己从小学画
第10章：她说自己是大学才开始接触艺术
→ 矛盾！
```

✅ **正确做法**：
- 人物设定一旦确立，所有后续内容必须符合
- 写作前查看人物档案
- 出现矛盾立即修正

## 📋 人物档案管理

### 主要角色档案内容

每个重要角色必须有清晰的档案：

```markdown
# 林深

## 基本信息
- 年龄：25岁
- 职业：大四学生，兼职商业咨询
- 家庭：工薪阶层，经济压力大

## 性格特点
- 核心：冷静、理性、有野心
- 优点：善于分析、执行力强、有底线
- 缺点：过于谨慎、不善表达情感
- 底线：不被钱收买、不失去原则

## 重要经历
- 第1章：在咖啡馆帮助沈云溪
- 第3章：与赵天启谈判，拿到50万赞助
- 第8章：发现被跟踪，警觉性提升

## 成长轨迹
- 初期：谨慎小心，试探性帮助
- 中期：逐渐展现能力，获得认可
- 后期：（待发展）

## 性格变化记录
- 无突兀变化
- 所有决策都符合"理性分析"的核心特质
```

## 🔗 伏笔追踪系统

### 什么是伏笔？

**埋下但未解释的内容**：
- 人物的神秘反应
- 未说明的背景
- 突然的态度变化
- 悬而未决的疑问

### 伏笔管理

**格式**：
```markdown
## 未解伏笔

### [章节号] 伏笔内容
- **埋下位置**：第X章
- **具体内容**：XXX对XXX有奇怪的反应
- **计划回应**：第X章揭晓
- **状态**：未回应 / 已回应

### 示例

**[第8章] 沈云溪的真实身份**
- 埋下位置：第1章
- 具体内容：手腕上的江诗丹顿手表，暗示她不是普通画家
- 计划回应：第8章通过林深调查揭晓（沈氏集团千金）
- 状态：✅ 已回应
```

## ⚠️ 写作前检查清单

**每次写新章节前，必须**：

1. [ ] 查看相关角色的档案
2. [ ] 确认性格特点和过去设定
3. [ ] 检查是否有未回应的伏笔
4. [ ] 确保行为符合角色逻辑

**写完后自查**：

1. [ ] 角色的行为是否符合性格？
2. [ ] 有没有突兀的变化？
3. [ ] 如果有变化，原因是否充分？
4. [ ] 新埋的伏笔是否已记录？

## 🎯 常见错误（灵活理解）

### 错误1：工具人化（并非绝对禁止）

**工具人可以存在**：
- ✅ 次要配角（服务员、路人、临时对手）可以是工具人
- ✅ 不是每个角色都需要复杂背景
- ❌ **主要角色**（林深、沈云溪、沈云瑶）不能工具人化

**例子**：
```
✅ 可以是工具人：
- 咖啡厅服务员
- 展览观众
- 临时出现的商业对手

❌ 不能工具人化：
- 林深（男主）
- 沈云溪、沈云瑶（女主）
- 赵天启（重要配角）
```

**原则**：根据角色重要性决定刻画深度，而不是一刀切。

### 错误2：性格分裂

❌ **坏例子**：
```
沈云溪：
第3章：单纯善良，完全信任林深
第5章：突然变得多疑，怀疑林深动机
（没有任何事件触发）
```

✅ **正确做法**：
```
第3章：单纯善良，完全信任林深
第4章：姐姐提醒她要小心
第5章：开始有所保留，但不是完全怀疑
```

### 错误3：伏笔遗忘

❌ **坏例子**：
```
第5章：暗示林深有个秘密
...
第20章：完全没提过这件事
```

✅ **正确做法**：
- 建立伏笔清单
- 每5章检查一次
- 确保重要伏笔都有回应

## 💡 灵活变通原则

### 规则不是教条

**记住**：
1. **规则是工具，不是枷锁**
   - 目的是保持质量，不是限制创作
   - 当规则妨碍故事流畅性时，优先故事

2. **懂得什么时候打破规则**
   - 主角能力 = 设定，无需解释
   - 次要角色可以工具人化
   - 剧情需要时可以适度简化

3. **人 vs AI 的区别**
   - AI：严格遵守所有规则
   - 人类：懂得根据情况变通
   - **目标**：像人类一样灵活写作

### 优先级排序

**什么时候严格遵守**：
1. 主要角色的核心性格（林深、沈云溪、沈云瑶）
2. 重要伏笔必须回应
3. 关键转折需要合理触发

**什么时候可以灵活**：
1. 次要配角的刻画深度
2. 主角技能的解释程度
3. 细节的严谨程度

### 判断标准

**问自己**：
- 这个规则是帮助故事，还是妨碍故事？
- 如果打破规则，会影响读者体验吗？
- 真人作者会怎么处理这个情况？

---

**核心理念**：用人类的灵活性写作，而不是机器的僵化执行！
