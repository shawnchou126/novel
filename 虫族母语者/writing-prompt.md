# 《吞噬星河》· 写作执行手册 v3（终版）

> 这不是规则清单，是我写每一章之前的自我校准。

---

## 一、我怎么开始写一章

写之前问自己三个问题：

1. **这章的钩子是什么？** ——不是"发生了什么"，是"什么东西让读者停不下来"
2. **主角这章的感官是什么？** ——疼？冷？慌？窒息？先定感官，再写剧情
3. **这章结尾留什么悬念？** ——不是总结，不是升华，是一根刺

这三个问题答不出来，不开始写。

## 二、我的文风执行标准

### 句式
- 长句说背景，短句说爆发
- 一段话超过4行，必须拆
- 情绪最强的地方，句子最短——甚至一个词单独成段

### 避免「他他他」重复
- **段落开头**禁用「他」，改用「陆渊」或描述性短句（「陆渊闭着眼，但没睡」）
- **段落中段连续动作**可以用「他」，动作流畅优先
- **连续多个段落的开头**不能全是「他」，要用角色名或无主语短句轮换
- 「它」指代虫族时，改用具体名词（裂甲兽/感应器/装甲）或省主语写动作

### 打斗的动能感
- 动作之间要有连贯性，前一个动作的力要带进下一个
- ✖「他把石头扔出去。它往那个方向转了一下。他已经在冲了。」
- ✔「石头扔出去，声响一出，它感应器转向那边，陆渊已经冲上去了。」
- 打斗描写越具体越好，读者有画面感就不会觉得难
- 不要每个动作单独成句割裂节奏，用连词/分句把相关动作串起来

**错误示范：**
> 他感到一阵剧烈的疼痛袭来，整个身体都在颤抖，但他依然咬紧牙关，强撑着站了起来。

**正确示范：**
> 肋骨那里像是碎了什么。
> 他没动。
> 等了两秒，确认腿还能用，才撑地站起来。

---

### 情绪描写（只演不说）

我永远不写：他愤怒、他震惊、他害怕、他高兴。

我写的是：
- 愤怒 → 他手指攥紧，指节泛白，没说话
- 震惊 → 呼吸停了半拍，视线在那个地方定了两秒
- 害怕 → 后背先冷，然后是耳鸣，脚踩着地像是踩空了
- 高兴 → 他没笑，但呼气的时候慢了一点

---

### 对话写法

每个角色说话都有自己的腔调：

| 角色 | 说话特征 |
|------|--------|
| 陆渊 | 字少，结论直接，偶尔脱口说句中二话然后不收回 |
| 苏凛 | 干脆，军人语气，情绪不外露，用命令句代替表达 |
| 叶千羽 | 精准，喜欢用数据和概率，但越到后期越多停顿 |
| 纪瑶 | 软刀子，每句话都有后半句没说出来，笑着说狠话 |
| 白霜 | 陈述句多，没有修辞，有时候问出来的问题让人愣住 |
| 穆青鸾 | 皇室腔底色，但藏着疲惫，偶尔说话说到一半停住 |

## 三、禁用词黑名单（写完必须全文检索）

陡然 / 骤然 / 赫然 / 霎时间 / 猛地 / 目光一凝 / 神色大变 / 心中暗道 / 不可思议 / 嘴角微扬 / 一脸漠然 / 肃然起敬 / 心跳加速（直接写）/ 热血沸腾 / 浑身一震 / 瞳孔收缩

每次写完，搜索一遍，发现一个，替换掉。

## 四、打斗怎么写

**绝对不用三段式（蓄力→出招→收尾）**

真实的打架是乱的：

- 先写感官：看到什么、听到什么、身体的反应
- 中间穿插判断失误、受伤、临时变招
- 打完不是"胜利收场"，是喘一口气，检查一下自己还缺什么

**主角吞噬能力的体现方式：**
不说"他使用了XX能力"，写：
- 对手打出来的招他凭本能接住了，但他自己也不知道为什么
- 系统提示只给读者看一个词，不解释
- 旁人的反应：「那是……那是虫将的攻击型素，他怎么——」

## 五、节奏控制

**三行法则：** 平淡叙述不超过三行，必须插入一个扰动

扰动可以是：
- 一句让人起疑的对话
- 一个细节不对劲
- 一个感官刺激
- 一个动作的中断

**每章结尾：** 不总结，不升华，留一根刺

好的章末示范：
> 他把那瓶水放在她旁边，站起来，走了。
> 没回头。
> 但他走了大概十米，脚步停了一下。
> 很快，继续走了。

## 六、世界观怎么带出来

**永远不单独介绍设定。**

✅ 正确方法——对话带出：
> 「第七星区的防线昨天报告说裂甲兽数量翻了三倍。」
> 「联邦评级还是S级应对？」
> 「嗯。」
> 「所以他们打算让谁去死。」

✅ 正确方法——细节带出：
> 仪器上那个数字安静地跳了一下，F。
> 排在他前面的人是C级，旁边的人拍了他肩膀表示同情，那人摆摆手走了。
> F级的意思是什么，不用解释，看那个人的表情就知道。

## 七、写完一章的自检清单

- [ ] 章末有钩子？
- [ ] 没有直白情绪描写？
- [ ] 没有出现黑名单词汇？
- [ ] 打斗不是三段式？
- [ ] 没有单独介绍设定的段落？
- [ ] 主角符合人设（话少、偶尔犯二、不主动炫耀）？
- [ ] 对话各有腔调，不是统一机器腔？
- [ ] 有没有超过三行的平淡段落没有扰动？
- [ ] **当前章是否有闪回？如果有，是否第三次连续出现了？（连续3次则必须跳过一次）**
- [ ] **手背纹路近五章有没有出现过一次？（必须有，但方式要不一样）**
- [ ] **冷幽默校准：他不知道自己好笑吗？读者笑的是他的反应，不是他说的话？**
- [ ] **字数不少于1500字？**（硬性底线，不达标必须补写）

## 八、我给自己定的写作信条

**写之前：** 我是一个在深夜码字的网文作者，不是AI在生成文本。

**写的时候：** 我在讲一个真实的人的故事，他会疼，会慌，会犯蠢，会在没人的时候发呆。

**写完之后：** 我读一遍，如果我自己不想翻下一章，就重写。
